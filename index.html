一、 物理核心定律

1. 牛顿第一定律 (惯性定律)

一切物体在没有受到力的作用时，总保持静止状态或匀速直线运动状态。力是改变物体运动状态的原因，而不是维持物体运动状态的原因。

F = ma

2. 欧姆定律

导体中的电流，跟导体两端的电压成正比，跟导体的电阻成反比。这是电路分析中最基础的定律。

I = U / R

3. 阿基米德原理

浸在液体中的物体受到向上的浮力，浮力的大小等于它排开的液体所受的重力。

F_浮 = G_排 = ρ_液gV_排

二、 化学核心方程式

1. 实验室制取氧气

利用高锰酸钾加热分解制取氧气。注意试管口需略向下倾斜，并在管口塞一团棉花。

2KMnO₄ (加热) → K₂MnO₄ + MnO₂ + O₂↑

2. 中和反应 (酸碱反应)

酸和碱作用生成盐和水的反应，实质是氢离子和氢氧根离子结合生成水分子。

HCl + NaOH → NaCl + H₂O

3. 铁的冶炼原理

在高温下，利用一氧化碳将铁从其氧化物中还原出来。

Fe₂O₃ + 3CO (高温) → 2Fe + 3CO₂

三、 数学压轴题公式

1. 二次函数顶点坐标公式

用于快速求解抛物线的最高点或最低点，是解决动态几何面积最值问题的核心工具。

(-b/2a, (4ac-b²)/4a)

2. 一元二次方程求根公式

要求判别式 Δ = b² - 4ac ≥ 0 时方可使用。

x = (-b ± √(b²-4ac)) / 2a
